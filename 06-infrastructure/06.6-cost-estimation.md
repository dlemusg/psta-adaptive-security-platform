# Estimación de Costos

## Propósito

Este documento presenta una estimación de costos de infraestructura para la Plataforma de Seguridad Transaccional Adaptativa (PSTA).

El objetivo es proporcionar una aproximación de inversión operativa (OPEX) asociada a la arquitectura propuesta y justificar las decisiones tecnológicas desde una perspectiva de costo-beneficio.

La estimación toma como referencia los requerimientos establecidos en el reto:

| Métrica | Valor |
|----------|---------|
| Eventos diarios | 80–120 millones |
| Pico transaccional | 8.000 req/s |
| Latencia autenticación | ≤ 200ms p99 |
| Latencia riesgo | ≤ 300ms p95 |
| Disponibilidad | 99.95% |
| RPO | ≤ 30 min |
| RTO | ≤ 15 min |
| Arquitectura | Multi-Zona Activo-Activo |
| Correlación histórica | 72 horas |

---
